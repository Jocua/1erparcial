Diferencia entre Hardware y Software
Los componentes de la computadora que tienen existencia física y que se pueden tocar son hardware, mientras que el software son aquellos programas que se ejecutan en el hardware. ... Por otro lado, para entregar su conjunto de instrucciones, el software necesita hardware.

HARDWARE

Conjunto de elementoos fisicos que constituyen una computadora o un sistema informatico

Hardware y Software

SOFTWARE

Conjunto de programas o rutinas que perimten a la computadore realizar determinadas tareas
